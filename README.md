# DE-Marketplace

DE-Marketplace is a fully client-side **Buy & Sell Marketplace** website built with HTML, CSS, and JavaScript. It allows users to browse products, add them to a cart, checkout with multiple payment options, manage their profile, and track orders. Sellers can upload and manage products. The website features a modern UI with a responsive full-screen slideshow, slide-out cart, and a blue-themed design.

---


   User Features**
- Browse and search products in the marketplace.
- Add products to a slide-out cart with quantity control.
- Remove items from the cart with a confirmation prompt.
- Checkout with multiple payment methods:
  - Credit Card
  - PayPal
  - Cash on Delivery
- Login and sign-up system with localStorage.
- Profile page:
  - View order history
  - Upload profile picture
  - Logout functionality

    Seller Features**
- Upload new products with name, price, and image.
- Edit or remove uploaded products via the dashboard.

   UI Features**
- Full-screen responsive slideshow on the home page.
- Slide-out cart that updates totals in real-time.
- Blue-themed design for home, cart, and upload pages.
- Responsive design for mobile and desktop.
